# Orbit Board
A local kanban board application powered by Electron, React and Tailwind. Supports multiple boards, drag &amp; drop, labels, due dates, dark mode, and offline autosave.

## Publish Updates
To publish a new update, run the following command:
```bash
$Env:GH_TOKEN="YOUR_TOKEN_HERE"
echo $Env:GH_TOKEN
npm run publish:win
```
